# code-pour-demain Code pour Demain

Code pour Demain

Apprendre aujourd’hui, construire demain.

Bienvenue

Bienvenue sur Code pour Demain, un projet citoyen né à San-Pédro pour offrir aux jeunes et aux femmes les clés du monde numérique.

  • 📌 Trouver un emploi
  • 📌 Créer leur propre activité
  • 📌 Accéder à l’éducation en ligne

À propos

Notre mission : rendre l’informatique accessible à tous.

Notre vision : créer une génération de citoyens autonomes, connectés et capables d’utiliser les outils numériques pour apprendre, entreprendre et transformer leur communauté.

Nos valeurs

  • ✅ Accessibilité
  • ✅ Solidarité
  • ✅ Innovation
